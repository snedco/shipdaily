# TheEntryLevelDevJobsAreD

> An app responding to: "The entry level dev jobs are disappearing."

This MVP responds to the trend "The entry level dev jobs are disappearing." picked up from r/webdev. It does one thing well.

## Getting started

```bash
npm install
cp .env.example .env.local
# Fill in Supabase credentials in .env.local
npm run dev
```

## Database

Run `schema.sql` in the Supabase SQL editor, or via `supabase db push` if using local Supabase.

## Deploy

```bash
vercel deploy --prod
```

Then add `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY` to Vercel env.

## Stack

- Next.js 14 (App Router)
- Supabase (Postgres + Auth + RLS)
- Tailwind CSS
- TypeScript
